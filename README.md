PictureQuiz
===

社内で頼まれた写真当てクイズをElectronで作ってみた。

Usege
---
**始めに**
```shell
$ git clone https://github.com/lightstaff/picture_quiz_electron.git YourApp
$ cd YourApp
$ npm install
```

**デバッグ**
```shell
$ npm dev
```

**パッケージング**

```shell
$ npm package
```
注:Windowsのみです
